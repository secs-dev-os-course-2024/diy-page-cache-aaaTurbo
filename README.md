## Описание Кода

Проект реализует пользовательский блочный кэш для работы с файлами, который использует политику вытеснения
Least Frequently Used (LFU). Кэш позволяет минимизировать задержки при чтении и записи данных, сохраняя часто
используемые блоки в памяти. Проект включает в себя API для работы с файлами, а также тестовые приложения для
демонстрации функциональности кэша.

▎BlockCache.h

Заголовочный файл, содержащий определения классов CacheBlock и BlockCache. Описывает структуру данных для хранения
блоков кэша и предоставляет интерфейс для работы с кэшем, включая методы для чтения, записи и управления блоками.

▎BlockCache.cpp

Реализация методов класса BlockCache. Содержит логику для добавления, поиска и удаления блоков кэша, а также для чтения
и записи данных в кэш и файл. Реализует политику LFU для управления кэшированием.

▎api.h

Заголовочный файл, определяющий API для работы с файлами. Включает функции для открытия, закрытия, чтения, записи,
перемещения указателя и синхронизации данных. Предоставляет интерфейс для взаимодействия с блочным кэшем.

▎api.cpp

Реализация функций API, определенных в api.h. Содержит логику для работы с блочным кэшем, включая инициализацию кэша,
обработку операций чтения и записи, а также управление файловыми дескрипторами.

▎main.cpp

Основной файл, который используется для тестирования функциональности блочного кэша. Генерирует файл с случайными
данными, помещает в него значение для поиска и использует API для чтения и записи данных, а также для демонстрации
работы кэша.

▎ema_search_int.cpp

Файл, который реализует логику поиска значения в файле с использованием API блочного кэша. Позволяет проверить, как кэш
улучшает производительность поиска, сравнивая время доступа к данным.

▎Использование

./ema_search_int
